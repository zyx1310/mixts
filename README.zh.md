# mixts

#### 项目简介

一个移动端（Android/iOS）的局域网文件传输应用，通过 Wi-Fi 在手机和电脑之间建立简单的 Web 文件服务，支持文件上传下载，类似于简易的 FTP/TFTP 网页版。

#### 功能特性

- 🌐 **局域网Web服务** - 手机开启服务后，电脑/其他设备可通过浏览器访问
- 📁 **文件管理** - 网页端查看、上传、下载 App 数据目录中的文件
- 🎮 **App控制** - 一键启停服务，显示访问地址和连接状态
- 🔒 **访问密码（可选）** - 默认不加密，可配置密码保护 Web 访问
- 🔒 **无需存储权限** - 使用 App 私有目录，无需申请额外权限
- 📂 **双目录存储** - 同时支持 App 私有目录和系统 Downloads/Mixts 目录，文件可通过文件管理器直接访问

#### 技术架构

| 平台 | 开发语言 | 最低版本 | 目标版本 | HTTP服务器 | UI框架 | 文件目录 |
|------|----------|----------|----------|------------|--------|----------|
| Android | Kotlin 2.0.x | API 26 (Android 8.0) | API 35 (Android 15) | NanoHTTPD 2.3.x | Jetpack Compose | App私有目录 |
| iOS | Swift 6.0.x | iOS 17 | iOS 18 | GCDWebServer 3.5.x | SwiftUI | Documents目录 |
